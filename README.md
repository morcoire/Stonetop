Stonetop character sheet for Roll20. Based on Stonetop by Jeremy Strandberg, CC BY-SA 4.0. Sheet customizations by Morcoire.

Use of these character sheets requires Roll20 Pro account in order to build a custom character sheet.

In order to build your custom character sheets, in your campaign home page select "Settings"> "Game Settings". 
  Scroll down to "Character Sheet Template" and select "Custom" in the drop down. 
  Copy the HTML code into the "HTML Layout" and the CSS code into "CSS Styling". You do need both sets of code for the character sheets to work.
